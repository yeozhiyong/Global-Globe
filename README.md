# Global Globe - NASA Edition

A cinematic 3D Earth visualization inspired by NASA imagery.

## Features

- 🌍 NASA Blue Marble textures with night lights
- ✨ Twinkling star field with depth layers
- 🌅 Day/night transition shader
- 💫 Atmospheric glow with rim lighting
- 🎥 Cinematic camera orbit
- 🎨 Dark grain processed aesthetic

## Live Demo

https://yeozhiyong.github.io/Global-Globe/

## Usage

Open `index.html` in any modern web browser.

## Technologies

- Three.js
- WebGL
- Custom GLSL shaders
